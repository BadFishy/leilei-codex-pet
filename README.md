## 安装

推荐直接将下面整段提示词复制到 Codex：

```text
请帮我直接安装这个 Codex 宠物“蕾蕾”，不要只告诉我安装步骤：
https://github.com/BadFishy/leilei-codex-pet

请下载或克隆该仓库，把仓库中的 leilei 文件夹安装到当前用户的 $CODEX_HOME/pets/leilei（Windows 通常是 C:\Users\<用户名>\.codex\pets\leilei）。如果已经存在同名目录，请先备份。安装完成后，确认目标目录中只有 pet.json 和 spritesheet.webp，并确认 pet.json 的 spriteVersionNumber 为 2。最后告诉我重新打开 Codex，并在宠物选择中选择“蕾蕾”。
```

如果 Codex 无法访问 GitHub，也可以手动下载仓库，将整个 `leilei` 文件夹复制到 `$CODEX_HOME/pets/` 下，再重新打开 Codex。

安装目录中应当只有：

- `pet.json`
- `spritesheet.webp`
