# 蕾蕾 · Codex Pet v2

## 安装

1. 解压分享包。
2. 将整个 `leilei` 文件夹复制到 `$CODEX_HOME/pets/` 下。
3. Windows 默认位置为 `C:\Users\<你的用户名>\.codex\pets\leilei`。
4. 重新打开 Codex，再选择“蕾蕾”。

安装目录中应当只有：

- `pet.json`
- `spritesheet.webp`
